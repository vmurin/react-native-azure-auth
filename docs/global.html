    validate(rules, values)

    Applies the validtion and transform rules to the given values

    Possible rules: rules = { parameters: { // define rules for needed keys from values input object realm: {} // just declared without rules, not required by default state: {required: true}, // required nonce: {required: false, type: 'string'}, // not required, but if given must have type strng clientId: {toName: 'client_id'}, // key name transform rule }, validate: true | false // if true (default) - pass through only declared params aliases: { connection: 'realm', clientID: 'clientId' } } Actions:

    • removes not declared in rules values (can be disabled by 'validate: false' parameter)
    • checks if all required values are exist in values ('required: true')
    • cheks if all given typed parameters have a required type
    • all values are optional by default (without "required" rule)
